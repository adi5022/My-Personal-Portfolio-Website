# Revision Pass 03

The site is moving in the right direction. This pass is focused on refinement and improving the user experience of the neuron interaction system. Do not redesign the entire section. Preserve what is currently working.

---

## 1. Telephone Design Revision

Ignore the custom telephone silhouette direction from the previous revision.

Revert to the simpler/vanilla telephone concept.

However:

* It must visually belong to the site's ecosystem.
* Match the visual language of the neuron.
* Match the line weights.
* Match the color palette.
* Match the SVG styling.

The telephone should feel like a natural extension of the environment rather than a separate object.

The goal is integration, not realism.

---

## 2. Increase Neuron Scroll Responsiveness

The scrolling behavior is much better now.

The movement feels connected to the user's actions.

However, the neuron path illumination should begin earlier.

Current issue:

Users are reaching sections before they fully experience the neuron activation.

Required changes:

* Increase path activation responsiveness.
* Start illumination sooner.
* Allow users to observe pathways lighting up while actively scrolling.
* Reduce the distance between scroll progress and visual feedback.

I want users to experience the activation process in real time.

The neuron should feel alive during the journey, not after the journey.

---

## 3. Popup Label Redesign

The popup label currently feels disconnected from the design.

Problems:

* It stands out too much.

Required changes:

* Redesign the popup styling.
* Make it feel native to the scene.
* Blend it into the site's visual language.
* Reduce visual harshness.
* Improve hierarchy.

The popup should feel discovered rather than injected.

## 4. Neuron Head Interaction Zone

Create a small interaction zone above the neuron head.

Requirements:

The interaction should feel intentional and rewarding.
